# AI Student Impact: Burnout Risk Prediction

This project implements a Decision Tree Classifier to predict three levels of student burnout risk (Low, Medium, and High) based on student behavioral data.

## Project Structure
The project is organized in a single root folder:
- `ai_student_impact_dataset (1).csv`  # The source dataset
- `final.ipynb`                        # The main analysis notebook
- `README.md`                          # This file

## How to Run
1. **Prerequisites:** Ensure you have [VS Code](https://code.visualstudio.com/) installed with the **Jupyter Extension**.
2. **Setup:** Ensure both the notebook and the dataset are in the same folder.
3. **Execution:** - Open `final.ipynb` in VS Code.
   - Select your Python kernel.
   - Click **"Run All"** to execute the pipeline. The script will handle the necessary package installations automatically.
